OTP(One-Time Pad)

This project comprises of three small programs written in C, that encrypt and decrypt information using a one-time pad like system. This project includes concepts such as multi-processing code, socket-based inter-process communication, unix, security, and more. This project was very fun to create and inspired me to learn more about security practices. Oregon State University requires this code to remain private. Watch demo video to view functionality.
